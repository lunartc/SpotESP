# SpotESP
A Spotify Integration with ESP32-S3 and a 3,2" TFT LCD Display ILI9488. Connected via a NodeJS acts as a Server between the Spotify API and LRCLIB API, then Connected to the ESP32-S3 via Serial.

First, you want to find your client id, client secret and also Spotify token for the Spotify API. WEB_APP_TOKEN are not required. ESP32_COM are required for connecting from the NodeJS to the ESP32-S3.
Spotify Token/API Can be found at developer page in Spotify.
